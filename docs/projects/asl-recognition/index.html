ASL Recognition

Kevin Richardson

Sign language recognition system for ASL Alphabet and ASL Words

Repo: https://github.com/kevinlrichardson-rgb/asl-recognition

Demo: https://huggingface.co/spaces/klrichardson/asl-recognition

  • MediaPipe Hands
  • Python
  • MLP
  • Gradio
  • Sign Language Recognition
  • Gesture Recognition
  • Deep Learning
  • LSTM
  • OpenCV
  • PyTorch

This project is a sign language recognition system that utilizes MediaPipe Hands for hand tracking and a Multilayer Perceptron (MLP) for classification. The system is designed to recognize American Sign Language (ASL) gestures, providing an accessible tool for communication between the deaf and hearing communities. The project is implemented in Python and leverages OpenCV for video processing, PyTorch for deep learning, and Gradio for creating an interactive web interface.
